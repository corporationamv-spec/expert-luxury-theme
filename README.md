Expert Luxury — Кастомная WordPress-тема
О проекте
Expert Luxury — это полностью кастомная WordPress-тема, разработанная с нуля без использования конструкторов страниц.
Проект реализован для премиального бренда в сфере beauty / body care и демонстрирует умение работать с архитектурой WordPress, динамическим контентом (ACF), кастомной логикой отображения и анимациями.

Тема построена с акцентом на:
чистую структуру кода
модульность
масштабируемость
производительность
аккуратный премиальный UI

Динамическая секция экспертов

Плавные fade-переходы
Независимая логика переключения
Адаптивная верстка
Анимации при скролле
Реализация через Intersection Observer
Плавное появление секций
Отсутствие конфликтов с логикой слайдера
Универсальный класс анимации
Архитектура темы
Разделение на template-parts
Логика вынесена из разметки
Корректная работа с get_field()

Подход к разработке
Проект построен без использования page builder’ов.
Вся логика реализована вручную.
Ключевые принципы:
Минимизация зависимостей
Чистый CSS без UI-фреймворков
Собственная реализация слайдера
Гибкость управления контентом через ACF
Версионный контроль через Git

Установка
Клонировать репозиторий:
git clone https://github.com/ВАШ_НИК/expert-luxury-theme.git

Поместить папку темы в:
wp-content/themes/
Активировать тему в админ-панели WordPress.
Установить плагин ACF и создать соответствующие поля.
